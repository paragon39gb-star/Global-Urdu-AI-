[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/[USER_NAME]/[REPO_NAME])

یہ ایپلیکیشن گلوبل ریسرچ سینٹر (GRC) کے تحت **قاری خالد محمود گولڈ میڈلسٹ** کا ایک تحقیقی شاہکار ہے۔ یہ ایک جدید تحقیقی انجن ہے جو قرآن، حدیث، فقہ اور سائنسی علوم پر مستند مواد فراہم کرتا ہے۔

## 🚀 براہِ راست چلانے کا طریقہ (Direct Run)

اگر Vercel کی لمٹ ختم ہو گئی ہے، تو آپ **Netlify** والا بٹن استعمال کریں:

### آپشن 1: Netlify (بہترین متبادل)
1. اوپر موجود **"Deploy to Netlify"** بٹن پر کلک کریں۔
2. اپنے GitHub اکاؤنٹ کو جوڑیں۔
3. **Environment Variables** میں جا کر `API_KEY` لکھیں اور اپنی کی (Key) پیسٹ کریں۔

### آپشن 2: Vercel
1. اوپر موجود **"Deploy with Vercel"** بٹن پر کلک کریں۔
2. اپنی `API_KEY` فراہم کریں اور 'Deploy' پر کلک کریں۔

## 💻 مقامی کمپیوٹر پر سیٹ اپ (Local Setup)

1. **Repository کلون کریں:**
   ```bash
   git clone https://github.com/[USER_NAME]/[REPO_NAME].git
   cd [REPO_NAME]
   ```
2. **لائبریریز انسٹال کریں:** `npm install`
3. **فائل بنائیں:** ایک نئی فائل `.env` کے نام سے بنائیں اور اس میں اپنی API Key لکھیں: `API_KEY=your_key_here`
4. **ایپ چلائیں:** `npm run dev`
